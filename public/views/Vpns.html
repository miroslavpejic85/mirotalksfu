Outline VPN клиенты

Выберите установщик для своей платформы, чтобы подключиться к защищённому Outline-серверу. Все ссылки ведут на официальные источники поставщика.

  • iOS

    Скачайте приложение в App Store и импортируйте ключ доступа, чтобы подключиться к серверу.

    Скачать в App Store

  • Android (Google Play)

    Установите Outline из Google Play для автоматических обновлений и удобного управления подключениями.

    Открыть в Google Play

  • Android (APK)

    Скачайте последнюю версию клиента напрямую в виде APK-файла и установите вручную.

    Скачать APK

  • Windows

    Установите десктопный клиент Outline и импортируйте конфигурацию. Файл скачивается в формате .exe.

    Скачать для Windows

  • macOS

    Используйте официальный клиент Outline для macOS из App Store.

    Скачать для macOS

После установки откройте приложение, выберите пункт «Добавить сервер» и вставьте свой ключ доступа. Он включает адрес сервера, порт, пароль шифрования и имя соединения — вся информация импортируется автоматически.
